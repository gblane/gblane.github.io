Measuring Changes in the Absorption Coefficient

Measuring Changes in the Absorption Coefficient

Here we describe six different methods for measuring absorption changes with the data collected in NIRS :

  • SD I: Capable in CW and FD with only one source and detector pair.

  • SD φ: Measurable in FD with only one source and detector pair.

  • SS I: Feasible in CW and FD with measurement at multiple ρs.

  • SS φ: Afforded by a FD with measurement at multiple ρs.

  • DS I: Realizable in CW and FD using a DS set (Chapter [ch:DSintro]).

  • DS φ: Possible in FD using a DS set (Chapter [ch:DSintro]).

Each of these methods yields a time trace of effective Δμa, which may then be converted into time traces of ΔO and ΔD (Appendix [app:recChrom]).

Generalized Total Optical Path-Length

Before we can discuss ways to calculate Δμa we must first define ⟨L̃⟩ which is utalized to determine various coeficents in the following sections. Using the R̃ from Equation [equ:RCom_semiInf] and its accompanying definitions of variables we can write ⟨L̃⟩ for a semi-infinite medium with extrapolated boundary: \[\begin{split} \acrshort{LpathCom}= \frac{3\acrshort{musp}}{8\pi\acrshort{RCom}}\\ \times\left(\frac{z_0 e^{-\|\acrshort{r}_1\|\acrshort{mueffCom}}}{\|\acrshort{r}_1\|}-\frac{z_0' e^{-\|\acrshort{r}_2\|\acrshort{mueffCom}}}{\|\acrshort{r}_2\|}\right) \end{split} \label{equ:Lcom_semiInf}\] which primarily depends on μa, μ′s, and ρ (through the r⃗s and R̃). In following sections ⟨L⟩ is written to depend on the spatial position of the source and detector, this is because following sections are more general than the semi-infinite medium. Here where the semi-infinite medium is considered only the ρ along the surface is of importance. Code for the ⟨L̃⟩ can be found in Listing [lst:Lpath].

⟨L⟩ can be of various data-types such as FD I (id est amplitude) or FD φ (\(\langle L \rangle_{I}\) or \(\langle L \rangle_{\phi}\)) which are obtained from: \[\acrshort{Lpath}_{\acrshort{I}}=\Re\left[\acrshort{LpathCom}\right]\] \[\acrshort{Lpath}_{\acrshort{phi}}=\Im\left[\acrshort{LpathCom}\right]\]

In the case of the ⟨L⟩ for CW I (\(\langle L \rangle_{\text{CW}}\)), the above expressions may still be used simply by setting ω or fmod to zero. This will result in a real instead of complex ⟨L⟩ which is itself \(\acrshort{Lpath}_{\acrshort{CW}}\). Unlike \(\langle L \rangle_{I}\) or \(\langle L \rangle_{\phi}\), \(\langle L \rangle_{\phi}\) has a clear physical meaning which is, the average length of the path of a photon that travels from the source to be detected by the detector. The meaning is no longer clear in FD since we now consider the phase and amplitude of photon density waves and ⟨L̃⟩ is complex. For this reason, we name ⟨L̃⟩ the general path-length since it is not clear that it represents a physical path despite having the units of length.

Absorption Changes Measured with Single-Distance Intensity or Phase

I changes measured with SD can be converted to changes in absorption using the modified Beer-Lambert Law (mBLL) and the DPF (denoted here with \(\text{DPF}_{I}\) to highlight its use in conjunction with intensity measurements). One may introduce a similar, generalized DPF concept for SD φ (denoted as \(\acrshort{DPF}_{\acrshort{phi}}\)). In the following expressions we also utilize ⟨L⟩ for I or φ (\(\acrshort{Lpath}_{\acrshort{I}}\) or \(\acrshort{Lpath}_{\acrshort{phi}}\)) in-place of DPF which result in more concise expressions since they do not require ρ.

Considering a baseline SD I or SD φ (\(\text{SD}I_0\) or \(\text{SD}\phi_0\)), Δμa may be expressed as follows: \[\begin{split} \Delta\mu_a_{,\text{SD}I}= \frac{\ln{\left(\frac{\text{SD}I_0}{\text{SD}I}\right)}}{\langle L \rangle_{I}}= \frac{\ln{\left(\frac{\text{SD}I_0}{\text{SD}I}\right)}}{\rho\text{DPF}_{I}}\\ \approxeq-\frac{\left(\frac{\text{SD}I-\text{SD}I_0}{\text{SD}I_0}\right)}{\rho\text{DPF}_{I}}= -\frac{\left(\frac{\text{SD}I-\text{SD}I_0}{\text{SD}I_0}\right)}{\langle L \rangle_{I}} \end{split}\] \[\begin{split} \Delta\mu_a_{,\text{SD}\phi}=-\frac{\text{SD}\phi-\text{SD}\phi_0}{\langle L \rangle_{\phi}}\\ =-\frac{\text{SD}\phi-\text{SD}\phi_0}{\rho\text{DPF}_{\phi}} \end{split}\] Since for 𝒴: \[\acrshort{DPF}_{\acrshort{Y}}=\frac{\acrshort{Lpath}_{\acrshort{Y}}}{\acrshort{rho}}\]

Example values of ⟨L⟩ and DPF for various optical properties can be found in the supplementary material of .

Absorption Changes Measured with Intensity or Phase Slopes

Changes in slopes measured with SS can also be converted into Δμa. In the case of I, the slope refers to the slope of ln(ρ²I) on ρ (\(\text{SS}I\)), while for φ the slope is of φ on ρ (\(\acrshort{SS}\acrshort{phi}\)), and we introduce baseline slopes (\(\acrshort{SS}\acrshort{I}_0\) or \(\acrshort{SS}\acrshort{phi}_0\)). For slope methods we also introduce the DSF which is a proportionality constant to convert to Δμa similar to DPF.

For Δμa from SS we have: \[\acrshort{dmua}_{,\acrshort{SS}\acrshort{I}}=-\frac{\acrshort{SS}\acrshort{I}-\acrshort{SS}\acrshort{I}_0}{\acrshort{DSF}_{\acrshort{I}}}\] \[\acrshort{dmua}_{,\acrshort{SS}\acrshort{phi}}=-\frac{\acrshort{SS}\acrshort{phi}-\acrshort{SS}\acrshort{phi}_0}{\acrshort{DSF}_{\acrshort{phi}}}\] Considering 𝒴 which can be either I or φ we can define DSF: \[\text{DSF}_{\mathcal{Y}}= \frac{\sum_{i=1}^{N_{\text{SD}}}{\left(\rho_i-\bar{\rho}\right)\langle L \rangle_{\mathcal{Y}}\left[\vec{r}_{src},\vec{r}_{det,i}\right]}}{\sum_{i=1}^{N_{\text{SD}}}\left(\rho_i-\bar{\rho}\right)^2}\] where we note that ⟨L⟩ is a function of the source and detector position (\(\acrshort{r}_{src}\) and \(\acrshort{r}_{det}\)). Here, \(N_{\acrshort{SD}}\) refers to the number of SD measurements in the SS set which is being considered. Additionally, \(\acrshort{r}_{det}\) is indexed by \(i\) since the expression assumes a common source and multiple detectors in the SS set, if the opisite is true the index would be on the source position. In general the sum is over the various SD sets that make up the SS and ⟨L⟩ depends on the spatial positions of the source and detector in the SD set. Next we will define the various ρ variables in the above expression: \[\acrshort{rho}_i=\|\acrshort{r}_{src}-\acrshort{r}_{det,i}\|\] \[\bar{\acrshort{rho}}=\frac{\sum_{i=1}^{N_{\acrshort{SD}}} \acrshort{rho}_i}{N_{\acrshort{SD}}}\] where the detector was chosen to be indexed as before, but this can be adapted depending on the particular SS arrangement.

Finally, we can define the Δμa for a DS set. Considering a DS set made of SSs \(\text{SS}_1\) and \(\acrshort{SS}_2\) with corresponding DSFs for those SS sets of \(\acrshort{DSF}_{\acrshort{Y},1}\) and \(\acrshort{DSF}_{\acrshort{Y},2}\) we can write Δμa for DS: \[\begin{split} \Delta\mu_a_{,\text{DS}I}=\\ -\frac{\left\{\text{SS}I-\text{SS}I_0\right\}_{\text{SS}_1}+\left\{\text{SS}I-\text{SS}I_0\right\}_{\text{SS}_2}}{\text{DSF}_{I,1}+\text{DSF}_{I,2}} \end{split}\] \[\begin{split} \Delta\mu_a_{,\text{DS}\phi}=\\ -\frac{\left(\text{SS}\phi-\text{SS}\phi_0\right)_{\text{SS}_1}+\left(\text{SS}\phi-\text{SS}\phi_0\right)_{\text{SS}_2}}{\text{DSF}_{\phi,1}+\text{DSF}_{\phi,2}} \end{split}\] for both DS I and DS φ.

Example values of DSF for various optical properties can be found in the supplementary material of .
